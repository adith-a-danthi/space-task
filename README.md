# Space

## Tech Stack

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- Vanilla CSS

---
## Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
